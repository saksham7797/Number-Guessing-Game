# Number Guessing Game (Java CLI)

A robust Command Line Interface (CLI) game developed in Java where the user guesses a random number within a limited number of attempts based on difficulty levels.

## Features
- **Difficulty Levels:** Easy (10 attempts), Medium (5 attempts), Hard (3 attempts).
- **Input Validation:** Handles non-integer inputs gracefully without crashing.
- **Session High Score:** Tracks the lowest number of attempts in the current session.
- **Time Tracking:** Displays the time taken to win a round.

## How to Run
1. Compile the file: `javac Main.java`
2. Run the game: `java Main`
